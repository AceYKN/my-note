# 3.11 語序——德語句子的基本结构

1.  **如果知道如何定位动词，阅读德语就会容易不少。一般而言，动词的人称形式（限定动词）在德语中会出现在以下三個位置之一：**
    * (a) 緊跟在主詞之後（正常語序 / Normal word order）
    * (b) 緊接在主詞之前（倒裝語序 / Inverted word order）
    * (c) 位於句子或從句的末尾（從屬語序 / Dependent word order）

2.  **這三種德語語序的基本形式如下：**

    * **(a) 正常語序 (Normal word order)：**
        主詞及其修飾語 + 動詞 + 限定動詞的修飾語
        * **Der alte Mann / geht / langsam in das Haus.** 
            （The old man / 慢慢地走 / 進房子裡。）

    * **(b) 倒裝語序 (Inverted word order)：**
        除了連詞或感嘆詞以外的任何單詞 + 動詞 + 主詞及其修飾語 + 其他修飾語
        * **Wann / kommt / Karl / nach Hause?**
            When is Karl coming home?（卡尔什么时候回家？）
        * **Gehen / Sie / bitte nach Hause!**
            Please go home!（请回家！）
        * **Den Hund / sehe / ich / auf der Straße.** 
            （那隻狗 / 我看見 / 在街道上。/ 我在街上看見那隻狗。）

    * **(c) 從屬語序 (Dependent word order)：**
        從屬連詞 + 主詞及其修飾語 + 其他修飾語 + 動詞
        * **... weil / Karl / nach Hause / kommt.**
            （……因為 / 卡爾 / 回家。）
        * **... wenn / Sie / nach Hause / gehen.**
            （……如果 / 您 / 回家。）
        * **... dass / ich / den
        * Hund / auf der Straße / sehe.**
            （……那隻狗 / 我看見 / 在街道上。/ ……我在街上看見那隻狗。）
