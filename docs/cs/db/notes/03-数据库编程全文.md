## DATEPART 函数

题面/课堂内容：

```text
SQL 函数 DATEPART
日期/时间函数，返回表示日期/时间表达式指定部分的值的整数。
格式：DATEPART(datepart, date-expression)
```

示例：

```sql
DECLARE @xs int, @xq int;

SELECT @xs = DATEPART(hh, GETDATE());
SELECT @xq = DATEPART(weekday, GETDATE());
```

方法：

- `DATEPART(hh, GETDATE())` 取当前小时。
- `DATEPART(weekday, GETDATE())` 取当前星期几。
- SQL Server 默认 `DATEPART(weekday, ...)` 与 `DATEFIRST` 设置有关；课堂题中按周一到周五判断通常写 `IN (2,3,4,5,6)`。

## 公司数据库触发器题

题面：

```text
Database programming
Consider the company database of Question I.

1. Write an SQL trigger to carry out the following action:
   The SaleDetail table is only permitted to be modified during business hours
   (9:00 AM - 5:00 PM) on weekdays (Monday-Friday).

2. Create a stored procedure with parameters, give the SalemanNO of a staff,
   display the product names he sold.
```

参考触发器：

```sql
CREATE TRIGGER trg_SaleDetail_business_hours
ON SaleDetail
AFTER INSERT, UPDATE, DELETE
AS
BEGIN
    DECLARE @hour int, @weekday int;

    SELECT @hour = DATEPART(hh, GETDATE());
    SELECT @weekday = DATEPART(weekday, GETDATE());

    IF NOT (@hour BETWEEN 9 AND 17 AND @weekday IN (2, 3, 4, 5, 6))
    BEGIN
        PRINT 'Not business hours!';
        ROLLBACK;
    END
END;
```

原图可见代码片段：

```sql
BEGIN
    DECLARE @hour int, @weekday int;
    SELECT @hour = DATEPART(hh, GETDATE());
    SELECT @weekday = DATEPART(weekday, GETDATE());

    BEGIN

        ROLLBACK;

```

参考存储过程：

```sql
CREATE PROCEDURE sp_sold_products
    @SalemanNO varchar(9)
AS
BEGIN
    SELECT PName
    FROM Product
    WHERE PNO IN (
        SELECT PNO
        FROM SaleDetail
        WHERE SalemanNO = @SalemanNO
    );
END;
```

## 员工姓名输出参数存储过程

题面：

```text
假设存在一个员工信息表 Employees(emp_id, emp_name, dept_id, salary)，
编写一个 SQL 存储过程。

存储过程名称：proc_get_employee_name
输入参数：in emp_id INT（员工 ID）
输出参数：out emp_name VARCHAR(50)（员工姓名）

功能要求：
若输入的员工 ID 存在，将该员工姓名赋值给输出参数；
若输入的员工 ID 不存在或为 NULL，输出参数赋值为“无此员工”。
```

参考写法，接近 MySQL/通用存储过程语义：

```sql
CREATE PROCEDURE proc_get_employee_name (
    IN in_emp_id INT,
    OUT out_emp_name VARCHAR(50)
)
BEGIN
    IF in_emp_id IS NULL THEN
        SET out_emp_name = '无此员工';
    ELSE
        SELECT emp_name
        INTO out_emp_name
        FROM Employees
        WHERE emp_id = in_emp_id;

        IF out_emp_name IS NULL THEN
            SET out_emp_name = '无此员工';
        END IF;
    END IF;
END;
```

SQL Server 写法：

```sql
CREATE PROCEDURE proc_get_employee_name
    @in_emp_id int,
    @out_emp_name varchar(50) OUTPUT
AS
BEGIN
    SELECT @out_emp_name = emp_name
    FROM Employees
    WHERE emp_id = @in_emp_id;

    IF @in_emp_id IS NULL OR @out_emp_name IS NULL
        SET @out_emp_name = N'无此员工';
END;
```

## 学生姓名输出参数存储过程

课堂代码：

```sql
IF EXISTS (
    SELECT name
    FROM sysobjects
    WHERE name = 'Sele_sname'
      AND type = 'p'
)
DROP PROCEDURE Sele_sname;
GO

CREATE PROCEDURE Sele_sname
    @in_sno char(6),
    @out_sname char(20) OUTPUT
AS
BEGIN
    SELECT @out_sname = sname
    FROM student
    WHERE sno = @in_sno;

    IF @out_sname IS NULL
        SET @out_sname = N'该学生不存在！';
END;
GO
```

调用示例：

```sql
DECLARE @OUTPUT_sname char(20);

EXEC Sele_sname
    @in_sno = '990101',
    @out_sname = @OUTPUT_sname OUTPUT;

SELECT @OUTPUT_sname AS 学生姓名;

DECLARE @OUTPUT_sname char(20);

EXEC Sele_sname
    @in_sno = '990191',
    @out_sname = @OUTPUT_sname OUTPUT;

SELECT @OUTPUT_sname AS 学生姓名;

DECLARE @OUTPUT_sname char(20);

EXEC Sele_sname
    @in_sno = NULL,
    @out_sname = @OUTPUT_sname OUTPUT;

SELECT @OUTPUT_sname AS 学生姓名;
```

## 带参数存储过程 Sele_staff

题面：

```text
编写带参数的存储过程 Sele_staff：
查询姓名第二个字为“武”的员工的部门名和负责人信息，
并以标签格式输出“职工号 姓名 部门 负责人”。
按 SQL Server 语法编写。
```

原图可见/校正后的代码：

```sql
IF EXISTS (
    SELECT name
    FROM sysobjects
    WHERE name = 'Sele_staff'
      AND type = 'P'
)
DROP PROCEDURE Sele_staff;
GO

CREATE PROCEDURE Sele_staff
    @pe_name varchar(10)
AS
BEGIN
    DECLARE Cur1 CURSOR
    FOR
        SELECT p.pno, p.pname, d.dname, d.mana
        FROM Pe AS p
        JOIN Dept AS d ON p.dptno = d.dptno
        WHERE p.pname LIKE '_' + @pe_name + '%';

    OPEN Cur1;

    DECLARE @p_no varchar(10),
            @p_name varchar(10),
            @dp_name varchar(10),
            @dp_mana varchar(10);

    FETCH NEXT FROM Cur1
    INTO @p_no, @p_name, @dp_name, @dp_mana;

    WHILE @@FETCH_STATUS = 0
    BEGIN
        SELECT 职工号 = @p_no,
               姓名 = @p_name,
               部门 = @dp_name,
               负责人 = @dp_mana;

        FETCH NEXT FROM Cur1
        INTO @p_no, @p_name, @dp_name, @dp_mana;
    END;

    CLOSE Cur1;
    DEALLOCATE Cur1;
END;
GO
```

说明：

- “姓名第二个字为武”在 SQL Server 中可写 `LIKE N'_武%'`。
- 原图中有 `Dname like '%' + @pe_name + '%'` 与 `sname like '%' + @stu_name + '%'` 类似课堂示例；针对“第二个字”更准确的是 `_武%`。

## 游标定义与循环

课堂示例：

```sql
DECLARE @stuSno varchar(10),
        @stuName varchar(10),
        @stuSex varchar(10),
        @stuSage varchar(10);

DROP PROCEDURE IF EXISTS Proc1;

DECLARE Cur_Student2 CURSOR
FOR
    SELECT sno, sname, sex, sage
    FROM student
    WHERE sname LIKE '%' + @stu_name + '%';

OPEN Cur_Student2;

FETCH NEXT FROM Cur_Student2
INTO @stuSno, @stuName, @stuSex, @stuSage;

WHILE @@FETCH_STATUS = 0
BEGIN
    SELECT 学号 = @stuSno,
           姓名 = @stuName,
           性别 = @stuSex,
           年龄 = @stuSage;

    FETCH NEXT FROM Cur_Student2
    INTO @stuSno, @stuName, @stuSex, @stuSage;
END;

CLOSE Cur_Student2;
DEALLOCATE Cur_Student2;
```

游标步骤：

1. 定义游标：`DECLARE cursor_name CURSOR FOR SELECT ...`
2. 打开游标：`OPEN cursor_name`
3. 读取当前行：`FETCH NEXT FROM cursor_name INTO ...`
4. 循环判断：`WHILE @@FETCH_STATUS = 0`
5. 关闭游标：`CLOSE cursor_name`
6. 释放游标：`DEALLOCATE cursor_name`
