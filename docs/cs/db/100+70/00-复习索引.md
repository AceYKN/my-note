复习建议：

1. 先看“答案”，再读“讲解”，最后遮住答案重新写一遍。
2. SQL 题不要背 Oracle 旧写法，重点记 SQL Server 的 `CREATE VIEW`、`UPDATE ... FROM`、`DROP INDEX index_name ON table_name`、`EXCEPT`、`INTERSECT`、双重 `NOT EXISTS`。
3. 范式题固定三步：先写函数依赖，再求候选码，最后判断部分依赖、传递依赖和 BCNF 条件。
